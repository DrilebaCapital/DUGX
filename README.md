## DUGX

> DUGX (Digital Ugandan Shilling) is an electronic currency running on the [Zilliqa network](http://zilliqa.com). Denominated in the Ugandan Shilling, it's primary goal is to facilitate electronic markets settled in Ugandan shillings.
